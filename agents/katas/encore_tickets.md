# Encore Tickets — Event Ticket Booking

Encore Tickets is an event ticketing platform. Customers browse events and reserve seats via a mobile app or website.

## Requirements

- Customers reserve seats for an event via mobile app or web.
- Payment must be confirmed before a seat reservation is finalized.
- Seat inventory must never be oversold — concurrent reservations for the same event must be safely resolved against remaining capacity.
- Once payment is confirmed and a seat is reserved, the system issues a digital ticket to the customer.
- Event organizers can update event listings, pricing, and seat capacity independently.
- Customers can cancel a reservation and request a refund before the event start time, releasing the seat back to available inventory.
- The platform tracks loyalty points for frequent attendees and applies rewards at checkout.
